# PR-3-HEXASOFTWARE-ANIMATINGING-WEBSITE🤖 AI Nova - Futuristic AI Landing Page

AI Nova is a modern and visually engaging AI-powered landing page developed using HTML, CSS, and JavaScript. The project focuses on creating a futuristic user experience through advanced animations, glassmorphism effects, interactive UI elements, and responsive design principles.

The website is designed to showcase an artificial intelligence platform with a premium SaaS-style interface. It combines modern web design trends with smooth animations to provide users with an immersive browsing experience.
